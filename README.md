# Automation Test Engineer – Training Repository

## 📌 Overview
This repository contains all my learning materials, practice exercises, assignments, and automation scripts developed during my **Automation Test Engineer training**

This repository acts as:
- A central place for all training-related commits
- A hands-on learning log for automation testing
- A reference for automation frameworks, tools, and best practices
- A growing codebase reflecting continuous improvement

---

## 🧪 Training Scope
The training focuses on both theoretical knowledge and practical implementation, including:

- Software Testing Fundamentals
- Manual Testing Concepts
- Automation Testing Basics to Advanced Topics
- Test Case & Test Scenario Design
- Automation Framework Development
- Debugging and Maintenance of Automation Scripts
- Version Control & Collaborative Development using Git

---

## 🛠️ Tools & Technologies
As part of the training, the following tools and technologies are used (based on training progression):

- Programming Languages: Java / TypeScript / JavaScript
- Automation Tools: Selenium / Playwright / Cypress
- Test Frameworks: TestNG / JUnit / PyTest
- Version Control: Git & GitHub
- IDEs: VS Code
- CI/CD Basics: GitHub Actions / Jenkins
